# personalEthCtrlpanel
个人用ETH/ETC控制面板

# Env requirement
php 7.1
nanopool.org

# Deployment
1. copy all files in src folder to your server.
2. edit config.json
3. run index.php

# how to use config profile
/index.php                     // <- This will set profile to default
/index.php&config=default      // <- You can specify profile here
